Detailed Tutorial: Advanced Network Enumeration with Nmap

This tutorial guides through a comprehensive, ethical scan on an authorized target (e.g., local lab VM). Permission is mandatory.

  1. Basic host discovery: nmap -sn 192.168.1.0/24 – Lists live hosts without port scanning.
  2. Port and service scan: nmap -sV -sC -O 192.168.1.100 – Detects versions, runs default scripts, fingerprints OS.
  3. Aggressive scan with output: nmap -A -T4 -oN scan_results.txt 192.168.1.100 – Includes traceroute, scripts; saves to file.
  4. Analyze results: Review open ports (e.g., 22/ssh, 80/http), services, and potential vulnerabilities from script output.
  5. Script-specific scan: nmap --script vuln 192.168.1.100 – Checks for known vulnerabilities.

Interpret results carefully and report responsibly. Reference: Nmap Documentation.
